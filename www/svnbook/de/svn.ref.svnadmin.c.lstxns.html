Name

svnadmin lstxns — Print the names of all uncommitted transactions.

Synopsis

svnadmin lstxns REPOS_PATH

Description

Print the names of all uncommitted transactions. See Abschnitt 5.4.3.2, „Entfernen unvollendeter Transaktionen“ for information on how uncommitted transactions are created and what you should do with them.

Examples

List all outstanding transactions in a repository:

$ svnadmin lstxns /var/svn/repos/
