Name

svnlook log — Print the log message, followed by a newline character.

Synopsis

svnlook log REPOS_PATH

Description

Print the log message.

Options

--revision (-r) REV
--transaction (-t) TXN

Examples

This shows the log output for revision 40 in our sample repository:

$ svnlook log /var/svn/repos/
